# common()

Computes the common (intersection) of all shapes or 2D geometries in the current context.

**Returns**: [**`Common`**](/docs/api/types/common)

:::tip
See the [guide](/docs/guides/booleans-and-fusion) for detailed usage examples.
:::

## Signatures

```ts
common(): Common
```

Computes the common (intersection) of all shapes or 2D geometries in the current context.

---

```ts
common(...objects: SceneObject[]): Common
```

Computes the common (intersection) of the given shapes or 2D geometries.

| Parameter | Type | Description |
|-----------|------|-------------|
| `...objects` | [SceneObject](/docs/api/types/scene-object)[] | The objects to intersect *(optional)* |
