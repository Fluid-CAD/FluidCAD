# slot()

Draws a slot with the given length and end radius.

**Returns**: [**`Slot`**](/docs/api/types/slot)

## Signatures

```ts
slot(distance: NumberParam, radius: NumberParam): Slot
```

Draws a slot with the given length and end radius.

| Parameter | Type | Description |
|-----------|------|-------------|
| `distance` | NumberParam | The slot length |
| `radius` | NumberParam | The end cap radius |

---

```ts
slot(targetPlane: SceneObject | PlaneLike, distance: NumberParam, radius: NumberParam): Slot
```

Draws a slot on a specific plane.

| Parameter | Type | Description |
|-----------|------|-------------|
| `targetPlane` | [SceneObject](/docs/api/types/scene-object) \| [PlaneLike](/docs/api/types/plane-like) | The plane to draw on |
| `distance` | NumberParam | The slot length |
| `radius` | NumberParam | The end cap radius |

---

```ts
slot(start: Point2DLike, distance: NumberParam, radius: NumberParam): Slot
```

Draws a slot from a start point with the given length and end radius.

| Parameter | Type | Description |
|-----------|------|-------------|
| `start` | [Point2DLike](/docs/api/types/point2dlike) | The start point |
| `distance` | NumberParam | The slot length |
| `radius` | NumberParam | The end cap radius |

---

```ts
slot(start: Point2DLike, end: Point2DLike, radius: number): Slot
```

Draws a slot between two cap-center points with the given radius.

| Parameter | Type | Description |
|-----------|------|-------------|
| `start` | [Point2DLike](/docs/api/types/point2dlike) | Center of the first cap |
| `end` | [Point2DLike](/docs/api/types/point2dlike) | Center of the second cap |
| `radius` | number | The end cap radius |

---

```ts
slot(geometry: SceneObject, radius: NumberParam, deleteSource?: BooleanParam): Slot
```

Creates a slot from a geometry edge with the given radius.

| Parameter | Type | Description |
|-----------|------|-------------|
| `geometry` | [SceneObject](/docs/api/types/scene-object) | The source geometry edge |
| `radius` | NumberParam | The end cap radius |
| `deleteSource` | BooleanParam | Whether to delete the source geometry (defaults to true) *(optional)* |

---

```ts
slot(targetPlane: SceneObject | PlaneLike, geometry: SceneObject, radius: NumberParam): Slot
```

Creates a slot from a geometry edge on a specific plane.

| Parameter | Type | Description |
|-----------|------|-------------|
| `targetPlane` | [SceneObject](/docs/api/types/scene-object) \| [PlaneLike](/docs/api/types/plane-like) | The plane to draw on |
| `geometry` | [SceneObject](/docs/api/types/scene-object) | The source geometry edge |
| `radius` | NumberParam | The end cap radius |

---

```ts
slot(targetPlane: SceneObject | PlaneLike, geometry: SceneObject, radius: NumberParam, deleteSource: BooleanParam): Slot
```

Creates a slot from a geometry edge, optionally keeping the source, on a specific plane.

| Parameter | Type | Description |
|-----------|------|-------------|
| `targetPlane` | [SceneObject](/docs/api/types/scene-object) \| [PlaneLike](/docs/api/types/plane-like) | The plane to draw on |
| `geometry` | [SceneObject](/docs/api/types/scene-object) | The source geometry edge |
| `radius` | NumberParam | The end cap radius |
| `deleteSource` | BooleanParam | Whether to delete the source geometry |
