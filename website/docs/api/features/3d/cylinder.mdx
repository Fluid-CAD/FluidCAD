# cylinder()

Creates a cylinder with the given radius and height.

**Returns**: [**`Transformable`**](/docs/api/types/transformable)

## Signatures

```ts
cylinder(radius: NumberParam, height: NumberParam): Transformable
```

Creates a cylinder with the given radius and height.

| Parameter | Type | Description |
|-----------|------|-------------|
| `radius` | NumberParam | The cylinder radius |
| `height` | NumberParam | The cylinder height |
