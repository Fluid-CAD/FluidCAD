# loft()

Creates a loft between two or more profiles.

**Returns**: [**`Loft`**](/docs/api/types/loft)

:::tip
See the [guide](/docs/guides/3d-operations/loft) for detailed usage examples.
:::

## Signatures

```ts
loft(...profiles: SceneObject[]): Loft
```

Creates a loft between two or more profiles.

| Parameter | Type | Description |
|-----------|------|-------------|
| `...profiles` | [SceneObject](/docs/api/types/scene-object)[] | The profiles to loft between (minimum 2) *(optional)* |
