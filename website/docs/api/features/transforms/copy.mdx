# copy()

[2D] Creates linear copies along an axis inside a sketch.

**Returns**: [**`SceneObject`**](/docs/api/types/scene-object)

:::tip
See the [guide](/docs/guides/copying) for detailed usage examples.
:::

## Signatures

```ts
copy(type: "linear", axis: AxisLike, options: LinearCopyOptions, ...objects: SceneObject[]): SceneObject
```

[2D] Creates linear copies along an axis inside a sketch.

| Parameter | Type | Description |
|-----------|------|-------------|
| `type` | "linear" | Must be `'linear'` |
| `axis` | [AxisLike](/docs/api/types/axis-like) | The axis to copy along |
| `options` | LinearCopyOptions | Copy count, spacing, etc. |
| `...objects` | [SceneObject](/docs/api/types/scene-object)[] | The objects to copy (defaults to last object) *(optional)* |
